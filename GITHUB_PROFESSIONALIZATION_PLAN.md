## 🔧 Configurações Recomendadas

### GitHub Organization Settings

1. **General**
   - [ ] Ativar "Require 2FA for organization members"
   - [ ] Ativar "Base permissions" como "Read"
   - [ ] Ativar "Member privileges" para "Allow members to create repositories"

2. **Member Privileges**
   - [ ] Permitir criação de repositórios públicos
   - [ ] Permitir criação de repositórios privados
   - [ ] Permitir criação de pages

3. **Repository Defaults**
   - [ ] Default branch: `main`
   - [ ] Default permissions: `Read`
   - [ ] Ativar "Require status checks to pass"

4. **Security**
   - [ ] Ativar "Require signed commits"
   - [ ] Ativar "Dismiss stale pull request approvals"
   - [ ] Ativar "Require code review before merging"
